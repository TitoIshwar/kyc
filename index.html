How It Works
① OCR Extraction
Tesseract.js scans the uploaded ID image and extracts all raw text from the document.
② LLM Parsing
Ollama (LLaMA 3) parses the raw text and returns a structured JSON with extracted_name and id_number.
③ Identity Match
The extracted name is decrypted from the DB (AES-256) and compared. Status is updated to Verified or Failed.
